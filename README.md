# Gestor de Tasques (To-Do List API)

API REST creada amb Flask i PostgreSQL per gestionar tasques.

## Funcionalitats
- Crear una tasca
- Llistar totes les tasques
- Consultar una tasca per ID
- Actualitzar una tasca
- Eliminar una tasca

## Requisits
- Python 3.8 o superior
- PostgreSQL
